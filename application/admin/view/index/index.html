说明:

1. 所有账号默认密码为：skyuse,为了您的账号安全，请点击右上角修改密码！

2. 默认超级管理员只有一个：甘文杰

3. 分组组长增加订单和分配提成

4. 员工都有查看的权限

5. 若要新增其他权限，请联系管理员



流程：

1. 超级管理员设置提成配置等级，比如，“设计部A级提成10%”，“设计部B级5%”

2. 分组组长添加订单，在订单完成后，根据具体情况选择提成配置。

3. 订单完成后，分组组长对订单提成分配到个人。

4. 设计部的普通员工，仅仅拥有查看订单，查看提成分配和查看个人提成。
